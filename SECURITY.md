# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in LLMFlow Search, do not open a public issue.

Report it privately by email:

- kazkozdev@gmail.com

Include a clear description of the issue, the affected component, reproduction steps,
and any suggested mitigation if available.

## Disclosure Process

Reported issues will be reviewed privately. Once the impact is understood, a fix or
mitigation path can be prepared before public disclosure.
